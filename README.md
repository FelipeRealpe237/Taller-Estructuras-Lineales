## 📖 Descripción del Taller

Este repositorio contiene la solución de un taller de programación en **Java**, enfocado en el uso de **estructuras de datos lineales** como arreglos, matrices y arrayList.

Durante el desarrollo del taller se implementan ejercicios que permiten practicar operaciones básicas sobre estas estructuras, como:

* Inicialización de arreglos y matrices
* Generación de números aleatorios
* Recorrido y búsqueda de datos
* Manipulación de información dentro de estructuras de datos
* Uso de arrayList para almacenar y gestionar datos dinámicos

El objetivo del taller es **fortalecer la lógica de programación y el manejo de estructuras de datos básicas en Java**.

```text
Taller-Estructuras-Lineales/
│
├── src/
```

**Descripción de carpetas**

* **arreglos/** → Ejercicios relacionados con arreglos unidimensionales.
* **matrices/** → Ejercicios que utilizan matrices y operaciones sobre ellas.
* **arraylist/** → Ejercicios que utilizan la estructura dinámica arrayList.

## Instrucciones para ejecutar el programa

1. Clonar el repositorio:

git clone https://github.com/FelipeRealpe237/TallerDeArregloUnidimensionalBidimensionalYArrayList.git

2. Abrrir el proyecto en un entorno de desarrollo Java, por ejemplo:

- NetBeans
- Visual Studio Code

3. Acceder a la carpeta:

``` src/ ```

Dentro de esta carpeta se encuentran tres subcarpetas que organizan los ejercicios del taller:

- **arraylist** → ejercicios utilizando ArrayList
- **arreglos** → ejercicios con arreglos unidimensionales
- **matrices** → ejercicios con matrices
  
4. Dentro de cada carpeta se encuentran las clases correspondientes a cada ejercicio.

5. Cada clase contiene un método `main`, desde donde se puede ejecutar el programa.

6. Ejecutar la clase del ejercicio que se desea probar para visualizar el resultado en la consola.
